# ops-toolbox

Small scripts for cleaning up local work folders.

## tf_clean.py

Clean Terraform local artifacts under the current folder (recursive).

The script only removes Terraform-generated dependencies and state files.
Terraform source files and outputs are kept in place.

### What gets removed

Directories:
- .terraform/
- terraform.tfstate.d/

Files:
- terraform.tfstate
- terraform.tfstate.backup
- terraform.tfstate.<timestamp>.backup
- .terraform.lock.hcl
- lock.hcl

### What is kept

- *.tf
- *.tfvars
- variables.tf
- any *.txt files (for example outputs generated by local_file)

### Usage (Windows)

Dry-run (no changes):
py tf_clean.py

yaml
Copy code

Apply and back up removed files before deleting:
py tf_clean.py --apply --backup

sql
Copy code

Apply and move removed files into a local trash folder:
py tf_clean.py --apply --trash

markdown
Copy code

### Notes

- Default mode is dry-run.
- Backup copies files to `_backup/tf-clean-<timestamp>/`.
- Trash mode moves files to `.tftrash/<timestamp>/`.
- Designed for repeated use across Terraform learning folders.
